## <span id="title_2">近期活動 _`最新消息`_</span>
### 公告

    顯示所有消息
    包含（活動、維修、贊助）

### 活動

    顯示活動消息
    限時活動、節慶活動、週年活動、特殊活動

### 維修

    顯示維修消息
    預告維修時間、維修完畢、維修內容、例行維修

### 贊助

    顯示贊助消息
    當有玩家贊助時會發文感謝
    或有贊助活動時也會在此公告
    並會定期向玩家公開展示贊助費用的用途
